# 算法可视化平台 - 单屏幕应用设计规范

## 1. 设计原则

### 1.1 单屏幕应用 (Single-Screen Application)
- **禁止滚动条**：所有内容必须在单个屏幕内完整展示
- **固定布局**：所有区域高度固定，不随内容动态变化
- **即时响应**：页面加载即展示所有区域，isRunning只控制内容填充

### 1.2 视觉层级
- 采用 **F型视觉动线**：从左到右、从上到下
- 重要信息 **左上优先**：输入区 → 可视化区 → 代码区
- 状态反馈 **底部固定**：控制面板始终可见

## 3. 组件设计

### 3.1 Header (56px)

**布局**：flex, justify-between, align-center, padding: 0 24px

```
[返回链接] [Logo/标题                        ] [算法思路] [GitHub Star]
```

**元素规范**：
- 返回链接：左侧，带箭头图标，文字"返回题库"
- 标题居中：题号 + 题目名称，字体18px, font-weight: 600
- 算法思路按钮：右侧，图标+文字，点击弹出Modal
- GitHub徽标：最右侧，显示Star数，带缓存机制

### 3.2 Input Panel (48px)

**布局**：flex, align-center, gap: 12px, padding: 0 24px

```
[输入: ______] [随机] [开始]    [示例: abcabcbb] [bbbbb] [pwwkew] ...
```

**元素规范**：
- 输入框：宽度200px，带字符计数，支持回车启动
- 随机按钮：生成随机字符串
- 开始按钮：主色调，突出显示
- 示例按钮组：横向排列，最多显示5个

### 3.3 Visualization Area (flex:1)

**布局**：position: relative, overflow: hidden

**空状态**：
- 居中显示引导文字："点击「开始」查看算法执行过程"
- 带箭头指向输入区域
- 背景使用微妙的网格图案

**数据状态**：
- D3.js SVG画布占满容器
- 支持拖拽平移、滚轮缩放
- 字符节点横向排列
- 滑动窗口用高亮背景表示

### 3.4 Code Area (360px)

**布局**：flex column, 固定宽度

```
┌──────────────────┐
│ [Java][Py][Go][JS]│  ← 语言切换 Tab (36px)
├──────────────────┤
│ 1  class Solution │  ← 代码区 (flex:1)
│ 2  ...            │
│                  │
├──────────────────┤
│ 变量: left=0 ...  │  ← 变量追踪区 (40px)
└──────────────────┘
```

**元素规范**：
- 语言切换：4个等宽按钮
- 代码区：行号 + 语法高亮 + 当前行高亮
- 变量追踪：底部固定，实时显示变量值

### 3.5 Control Panel (80px)

**布局**：flex column, padding: 12px 24px

```
[━━━━━━━━━━━━━━━━━━━━●━━━━━━━━━━]  步骤 12/34  (进度条)
[重置] [←] [▶] [→]          1.0×  (控制按钮)
```

**元素规范**：
- 进度条：可拖拽，绿色填充，高度8px
- 控制按钮：重置、上一步、播放/暂停、下一步
- 速度选择：下拉选择，0.5x / 1.0x / 1.5x / 2.0x
- 步骤显示：当前步骤 / 总步骤

## 5. 交互规范

### 5.1 空状态 → 数据状态

**原则**：
1. 页面加载即展示所有区域（空状态）
2. 点击开始按钮后，数据填充到现有区域
3. 区域高度和位置保持不变
4. 控制按钮从禁用变为启用

### 5.2 动画效果

- 算法步骤切换：300ms ease-in-out
- 指针移动：200ms ease-out
- 字符高亮：150ms ease
- 代码行高亮：即时，带背景色过渡

### 5.3 键盘快捷键

- ←：上一步
- →：下一步
- 空格：播放/暂停
- R：重置

## 6. 响应式断点

### 6.1 桌面 (≥1280px)
- 代码区域：360px
- 画布区域：flex:1

### 6.2 平板 (≥768px)
- 代码区域：280px
- 画布区域：flex:1

### 6.3 移动 (<768px)
- 画布区域：100%高度 60%
- 代码区域：100%高度 40%
- 垂直堆叠布局

## 7. 验收标准

### 7.1 功能验收
- [ ] 页面无水平滚动条
- [ ] 页面无垂直滚动条（100vh内完整展示）
- [ ] 所有区域默认可见（空状态）
- [ ] 开始按钮只填充数据，不改变布局
- [ ] 代码区语法高亮正常
- [ ] 画布可拖拽、可缩放
- [ ] 进度条可拖拽跳转

### 7.2 视觉验收
- [ ] 配色符合规范
- [ ] 字体符合规范
- [ ] 间距符合规范
- [ ] 动画流畅
- [ ] 无明显视觉断层
